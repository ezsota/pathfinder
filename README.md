# Pathfinder | Spacecraft Rentals Project
Pathfinder is a modern e-commerce single-page application project, built primarily using React Router and Bootstrap v5. 
This SPA project was built to emulate car rental sites, allowing authorized users to browse, filter, and rent available crafts.
The main objective of this project was to build an e-commerce site that successfully handles dynamic routing, responsive design, and user authentication.

# Tech Stack
- **Frontend:** React Router, React
- **Styling:** Bootstrap, CSS, PopperJS
- **Hosting:** Vercel
- **Version Control:** Git/GitHub

# Features
- Persistent user and reservation data
- Filter component utilizing craft details
- User authentication / Protected Routes
- Account and rental management
- Responsive and modern design
